# Research Idea Tracker

A simple desktop application for managing and tracking research ideas, built with Electron.

## Features

- Create, edit, and organize research ideas
- Markdown support with KaTeX for mathematical expressions
- Clean and intuitive user interface
- Data persistence with local JSON storage

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Build

```bash
npm run build
```
